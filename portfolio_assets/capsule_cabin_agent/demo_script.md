# 90 秒演示讲稿

这是一套基于瑞泽·海度假民宿已建项目复盘整理的规则型 AI PoC。它不声称原项目建设时已经使用 AI，而是展示如果后续同类装配式民宿扩建引入 AI，可以如何辅助早期配置和交付规划。

首页展示真实案例背景，以及卡素产品体系下约 38㎡单体和约 56㎡拼装两类产品。当前 PoC 固定在“瑞泽·海度假民宿 38㎡单体扩建”。

进入对话页后，我可以直接选择快捷模板：基于瑞泽·海度假民宿复盘，为滨海度假区新增一套 38㎡卡素单体产品，自助入住，配置基础智能控制。空间配置 Agent 会基于案例规则输出推荐产品、空间模块、标配/选配、智能设备、推荐依据和数据来源。

确认配置后，继续生成交付规划。交付规划 Agent 会把方案拆成 7 个阶段：设计确认、工厂加工/预制、运输、现场基础、吊装、拼接安装、调试验收。每个阶段都有任务、验收点和风险提示，并突出工厂加工/预制、拼接安装、调试验收这三个高风险环节。

最后在方案页可以看到完整配置与交付看板。这个 PoC 的重点不是替代专业设计或工程管理，而是把真实项目经验结构化成一个可交互、可解释、可演示的 AI 产品闭环。
