# 简历表述

- 基于真实装配式民宿案例，设计并实现回溯式 AI 配置与交付规划 PoC，覆盖需求输入、产品选型、空间配置、智能设备建议和交付阶段拆解。
- 复用既有 Vue 3 + FastAPI + LangGraph 项目骨架，将原搬家/装修 Agent 以最小改造方式替换为空间配置 Agent 与交付规划 Agent。
- 设计规则型 PoC 模式和项目复盘规则兜底，在 LLM/RAG 不可用时仍可完成作品集演示闭环。
- 将产品规格、空间模块、交付阶段和风险提示结构化为可展示的方案页与交付看板，用于支持面试场景中的业务建模说明。
- 明确区分 PoC、规则模拟与真实工程能力边界，避免将示例配置误表述为正式报价、设计或施工承诺。
