# 案例背景与 PoC 定位

瑞泽·海度假民宿是位于威海滨海度假区的营地型度假民宿，采用卡素产品体系 / Capsule Cabin。已知项目共 13 套客房，其中 7 套为约 38㎡单体产品，6 套为约 56㎡拼装产品。

本作品集项目不是声称原项目建设阶段已经使用 AI，而是基于项目参与经验、产品资料图、户型图与现场交付复盘进行回溯式产品化抽象，验证“业主需求输入 → 产品选型 → 空间配置 → 智能设备建议 → 交付任务规划”的 AI PoC 闭环。

固定案例为“瑞泽·海度假民宿 38㎡单体扩建（规则型 PoC）”，产品体系为卡素 Capsule Cabin，运营方式为偏无人化运营、自助入住 + 基础智能控制。

边界声明：配置、预算和周期均为示例，不构成正式设计、报价、施工组织或工程承诺。
