# 38㎡ Demo 案例

## 输入

基于瑞泽·海度假民宿复盘，为滨海度假区新增一套 38㎡卡素单体产品，自助入住，配置基础智能控制。

## 空间配置输出

- 推荐产品：Capsule Cabin S
- 推荐面积：约 38㎡
- 客群：不限定客群，空间紧凑，最多 4 人入住
- 风格：柔和粉白色系，海滨轻度假
- 运营方式：自助入住 + 基础智能控制

## 空间模块

- 睡眠区
- 卫浴区
- 休闲区
- 茶吧/收纳区
- 入口与行李区

## 配置清单

- 1.8m 双人床与床头阅读灯
- 整体卫浴模块
- 双人休闲座椅与小圆桌
- 迷你茶吧、冰箱位与开放收纳
- 入口换鞋与行李暂存区
- 粉白色系软装、耐潮墙面与易清洁地面

## 智能设备

- 智能门锁
- 基础照明控制
- 空调控制
- 烟感/水浸提醒
- 网络预留

## 交付阶段

1. 方案确认
2. 工厂预制与集成
3. 运输准备
4. 现场基础检查
5. 吊装与模块安装
6. 室内/智能设备调试
7. 验收与开业准备

## 风险提示

- 滨海环境需关注防腐、防水、通风与设备保护。
- 现场基础与吊装条件需由专业人员确认。
- 运输尺寸与道路条件需提前复核。
- 智能设备依赖网络、供电与现场调试。
- 消防、结构、安全和审批要求不由 PoC 自动确认。
