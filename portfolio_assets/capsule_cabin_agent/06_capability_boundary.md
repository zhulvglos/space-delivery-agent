# 能力边界

## 已实现

- 规则型 PoC 模式，无需登录即可访问。
- 固定 38㎡瑞泽·海度假民宿扩建案例。
- 空间配置 Agent：产品推荐、空间模块、配置清单、智能设备、示例预算说明。
- 交付规划 Agent：7 阶段、任务、验收点、风险提示。
- 前端本地案例规则兜底：后端不可用时仍可展示 PoC。
- 作品集页面与截图入口。

## 规则模拟

- 产品推荐基于关键词和固定业务规则。
- 预算仅表达配置层级，不输出真实金额。
- 交付阶段为示例拆解，不代表真实工期。
- 风险提示为通用交付关注点，不替代专业评审。
