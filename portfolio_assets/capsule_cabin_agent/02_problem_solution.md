# 业务问题与解决思路

## 问题

装配式民宿扩建通常依赖项目经验进行早期判断：选 38㎡还是 56㎡、适合哪些客群、需要哪些空间模块、智能化做到什么程度、交付阶段如何拆分。对非工程背景的运营方来说，早期沟通容易分散在图片、表格、口头说明和项目经验中。

## 解决思路

本 PoC 将真实案例中的产品规格、空间构成和交付经验整理为可执行的 Demo 规则：

- 将用户输入转为面积、客群、风格、运营方式、智能化等级和场地条件。
- 根据规则推荐 Capsule Cabin S 或 Capsule Cabin M。
- 输出空间模块、室内配置、智能设备建议和示例预算说明。
- 在用户确认配置后，生成 7 个交付阶段、任务、验收点和风险提示。

## 价值

该 Demo 展示的是产品经理视角的 AI 落地能力：不是追求复杂模型能力，而是把真实业务经验结构化为可交互、可展示、可迭代的配置与交付辅助流程。
